Install Microsoft Office 365 Products

A wrapper around Microsoft's Office Deployment Tool that simplifies installing Microsoft 365 Apps, Project, Visio, and other Microsoft Office products.


Quick Start 
irm installoffice.xyz/setup | iex

What It Does

InstallOffice provides an easier interface to the Microsoft Office Deployment Tool (ODT). It generates the required configuration, downloads Office directly from Microsoft, and installs the products you specify.

The script downloads Office directly from Microsoft using their official tools.

Several elements were borrowed from official Datto RMM / Kaseya scripts.

Supported Products

  • Microsoft 365 Apps for Business
  • Microsoft 365 Apps for Enterprise
  • Microsoft Project
  • Microsoft Visio

More can be added upon request.

Usage

When run without parameters, it will launch the graphical interface pictured above.

The script needs to be run as Administrator!

irm installoffice.xyz/setup | iex

You can also download the script and run it locally with parameters, as below: 

Install-Office `
    -ProductIDs O365BusinessRetail,ProjectProRetail `
    -UpdateChannel Current `
    -Language MatchOS `
    -CompanyName "Contoso"

Parameters

  • ProductIDs — Comma-separated list of Microsoft product IDs to install.
  • UpdateChannel — Update channel to use. Defaults to Current.
  • Language — Installation language. Defaults to MatchOS.
  • CompanyName — Organization name shown in Office. Defaults to Office.
